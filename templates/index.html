Frank Ruan's OpenPGP KeyServer

This site is Frank Ruan's OpenPGP KeyServer written in Python with Flask. It is not compatible with GPG nor PGP the program itself.

I cannot find any useful information about the protocol which GPG uses (aka. HKP(S)), thus I cannot make a backwards compatible server.

Contact me if you have found the detailed specifications about HTTP Key Protocol.

To use this keyserver, you will need to use our home-made script written in Python. (Prebuilt binary by PyInstaller is available)

https://github.com/orange2008/YetAnotherOpenPGPKeyServer

Clone the entire repo and you will find the 

yapks_client
file in it. Download it and move it to your binary path.

Alternatively, you can use the Releases, which is prebuilt.
